# 🍽 食事予約自動化スクリプト（Tampermonkey用）

このプロジェクトは、寮の食事予約サイトにおいて、
朝食・夕食の予約を自動化するためのTampermonkeyスクリプトです。

---

# ⚠️ 注意（重要）

このスクリプトは個人利用目的で作成されたものであり、
利用するサイトの利用規約に違反しない範囲で使用してください。

---

# 💡 Tampermonkeyとは？

## ■ Tampermonkeyとは
Tampermonkeyは、ブラウザ上でユーザースクリプトを実行できる拡張機能です。

以下のようなことが可能になります：

- Webページの自動操作
- ボタンの自動クリック
- DOM解析と自動入力
- 業務の自動化

---

---

## ■ 使い方

1. Tampermonkeyをインストール
2. 「ダッシュボード」を開く
3. 「新規スクリプト」を作成
4. `meal-auto.user.js` の中身を貼り付け
5. 保存
6. 対象サイトにアクセスすると自動実行(拡張機能から対象スクリプトをONにしてください)
